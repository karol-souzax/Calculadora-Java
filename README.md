# Calculadora-Java
Calculadora feita em java

Este projeto consiste em uma calculadora simples com interface gráfica, desenvolvida em Java, capaz de realizar operações matemáticas básicas como soma, subtração, multiplicação e divisão.

O objetivo é demonstrar conceitos de:

Programação orientada a objetos
Interface gráfica (GUI)
Manipulação de eventos

#Funcionalidades
➕ Soma
➖ Subtração
✖️ Multiplicação
➗ Divisão
🧮 Interface gráfica interativa
⚠️ Tratamento de erros (ex: divisão por zero)
